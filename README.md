Mini Inverter – 12V to 240V DC-AC Converter PCB
This project focuses on the design and fabrication of a compact DC to AC inverter capable of converting 12V DC battery input into 240V AC output for low-power applications.

Features:
12V DC input (Battery powered)
240V AC output
Oscillator circuit (IC 4047 / SG3525 / Arduino SPWM – mention yours)
MOSFET-based switching stage
Step-up transformer
Heat sink protection for MOSFETs
Compact single-layer PCB design
Short circuit / fuse protection (if added)

Tools Used:
KiCad / Eagle / Altium Designer
Oscilloscope for waveform testing
Multimeter for voltage measurement
PCB fabrication & soldering

Working Principle:
The oscillator generates alternating pulses that drive power MOSFETs in a push-pull configuration. The transformer steps up 12V AC (generated by switching) to 240V AC output.

Applications:
Emergency backup systems
Portable power supply
Rural electrification support
Small appliances (fan, light, charger)
